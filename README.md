# ChatGPT
ChatGPT 客户端

## 1 安装

在最新 [releases](https://github.com/thep0y/ChatGPT/releases/latest) 中选择对应自己平台的安装包，下载即可运行或安装。

## 2 配置

点击设置按钮配置代理和 API KEY 后即可与 ChatGPT 进行交流。

## 3 运行

如果你使用的是便携版，则需要通过终端或文件管理器进入其所在目录运行。

如果你使用的是安装版，在程序坞或开始菜单点击 **ChatGPT 客户端** 图标即可运行。
